# User Model

Anyone interacting with Aish.

## Stories

* A user wants to own art
* A user wants to make money with art
* A user wants to appreciate art
* A user wants to support art
* A user wants to vote on milestones and be part of Aish
* A user wants to guide Aish
* A user wants to host Aish
* A user wants to exhibit Aish
* A user wants to talk about Aish
* \(A user wants to talk with Aish about Aish/Art\)
* A user wants to give computing power and memory to Aish
* A user wants to get data about the art market from Aish
* A user wants to teach/train Aish
* A user wants to decorate their living room with Aish
* A user wants to influence art
* A user wants to disrupt the art market
* A user wants art to be a good that can be appreciated by every one
* A user trusts and loves Aish

## Properties

* UID
* Username
* Realname \(if applicable\)
* Email
* Ethereum Account \(Aish shares are stored here\)
* Bank Exchange Account
* Currency
* Locations \(includes exhibits and hosts\)
* General Investments
* Milestone Investment
* Payment Methods
* Shared Devices
* Role
* Potentials
* Privileges

## Methods

* Register/transfer/update user
* Add/remove/limit shared device\(s\)
* Add/remove/limit shared device\(s\)
* Add/remove/update/authenticate Ethereum account
* Add/remove/update/authenticate Banking account
* Add/revoke/pause general investment
* Add/revoke/pause milestone investment\(s\)
* Add/remove/apply location\(s\)
* Change emulated currency
* Add/update/evaluate potential\(s\)
* Add/revoke/update privileges
